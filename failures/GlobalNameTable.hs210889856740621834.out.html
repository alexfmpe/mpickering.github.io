-- | Global tranlsation table from Template Haskell function and

--
-- (function, constant constructor name, (arity, function to obtain a

  ('V.take          , (2, genExprFCall2L takeId                          ) ),

  ('V.shiftl        , (2, genExprFCall2L shiftlId                        ) ),
